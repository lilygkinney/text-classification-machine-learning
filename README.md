## Project Overview  

This project explores and compares multiple machine learning algorithms for text classification tasks. The goal was to evaluate how different predictive models perform on real-world datasets and to understand the strengths and limitations of each approach.  

The project was conducted as part of Florida State University's Directed Reading Program and focuses on applying core concepts from machine learning, including classification techniques, model evaluation, and performance comparison.  

---

## Objectives  

- Apply machine learning models to real-world text classification problems  
- Compare performance across multiple algorithms  
- Understand differences between parametric and non-parametric models  
- Evaluate model effectiveness using standard metrics  
- Gain practical experience in building and analyzing predictive models  

---

## Datasets  

The project uses two text-based datasets:  

- **IMDb Dataset** – Sentiment classification of movie reviews (positive vs. negative)  
- **Spam Dataset** – Classification of messages as spam or non-spam  

---

## Methods & Models  

The following machine learning models were implemented and compared:  

### Parametric Models
- Logistic Regression  

### Non-Parametric Models
- K-Nearest Neighbors (KNN)  
- Decision Tree  
- Random Forest  

### Additional Techniques
- Text preprocessing and vectorization  
- Feature extraction using techniques such as TF-IDF  
- Train-test split and cross-validation  
- Hyperparameter tuning

## Model Evaluation  

Models were evaluated using standard classification metrics:  

- Accuracy  
- Precision  
- Recall  
- F1 Score

## Key Insights  

- Different models perform better depending on the dataset and feature representation  
- Simpler models like logistic regression can perform competitively with more complex models  
- Ensemble methods such as random forests often improve predictive performance  
- Proper preprocessing and feature engineering are critical for text classification tasks
